# i2i-Academy-ApacheIgnite-1

This project demonstrates the usage of Apache Ignite 3 as an In-Memory Data Grid (IMDG).

## Features

- Apache Ignite 3 Thin Client connection
- Subscriber table creation
- Table cleanup before each execution
- Insertion of 5 dummy subscribers
- Random usage simulation
- Database updates using SQL
- Final state printing

## Technologies

- Java 17
- Maven
- Docker
- Apache Ignite 3.0.0
